A web-based Spotify clone built with **HTML, CSS, and JavaScript**. This project mimics the look and feel of the Spotify Web Player, allowing users to browse playlists, play songs, and manage a music library with an interactive interface.


---

## ðŸŒŸ Features

- **Dynamic Playlists:** Load multiple playlists like "Happy Hits" or "Sad Hits".  
- **Audio Playback:** Play, pause, skip to next/previous tracks.  
- **Song Library:** Browse all songs with album covers and song information.  
- **Real-time Search:** Filter songs in your library using a search bar.  
- **Seekbar:** Navigate within a song using a progress bar.  
- **Volume Control:** Adjust volume with an interactive slider.  
- **Responsive Design:** Works on desktop and mobile devices.  
- **Interactive UI:** Smooth animations and intuitive buttons for a better user experience.
